Exercise 3.1

In this exercise we are going to review basic XHTML syntax.

Text

XHTML is about content, not format. That's why <strong /> is used instead bold. It's the navigator the one that decide how to render it. If you want to write some italics, just emphasize it with <em />.

A new line in source code doesn't means a new line in rendered text. Although, with <br />
we can force a new line in rendering.
